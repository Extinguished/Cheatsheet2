HTML Table Tips

Tables in HTML

Element Description
<table> Begins a table
<tr> Begins a row
<th> Begins a cell that can be styled seperately
<td> Begins a cell
colspan="" Is a modifier that changes how many cells in a column it will use.
rowspan="" Is a modifier that changes how many cells in a row it will use.

HTML Tips

Common Elements in HTML

Element Description
<html> Want anything to work at all? Better have this tag encompassing the entire html file.
<div> A basic container. Very versatile but extremely generic.
<p> Begins a paragraph
<em> Italizes words between tags
<strong> Bolds words between tags
<img src=""/> Adds an image with src equal to location of image. Add "alt=" to provide text for an unloaded image. (Self closing)
<a> Stands for anchor. Used to add links to webpages by adding href="url" inside the opening tag.
<br> Line break!
<ul> Unordered list, alternatively you could use <ol> if you would like it numbered.
<li> List item.
<video src="" width="" height="" controls> Adds a video to the page. Ensure to have controls or autoplay. Is not a self closing tag.

CSS Tips

CSS

Command Description
!important Overrides all selectors. Be wary of use.
#idname Selects an id.
.classname Selects a classname.
elementname An element name by itself will select that element to style.
padding Creates extra space within an element.
margin Creates extra space around an element.
